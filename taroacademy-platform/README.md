# TaroAcademy Platform MVP

Това е базов проект за клиентски достъп до видео уроци и прост CRM.

## Страници

- `index.html` - клиентска зона. Клиентът вижда само вход с имейл.
- `admin.html` - CRM за добавяне на клиенти и Vimeo уроци.

## Как работи в този MVP

Данните се пазят в браузъра чрез `localStorage`.

Ако checkout системата може да пренасочва с имейл, използвай:

`index.html?email=client@email.com`

Ако този имейл е активен в CRM-а, клиентът ще влезе директно.

За реална платформа следващата стъпка е база данни и автоматично добавяне на имейла след checkout.
